### 📌 Description
Ce projet simule un feu de signalisation routière (Vert, Jaune, Rouge) piloté par un Arduino. Pour rendre le système plus interactif, un écran LCD 16x2 affiche des instructions en temps réel pour les usagers : 
* **"GO"** lorsque le feu est vert.
* **"STOP"** lorsque le feu est jaune ou rouge.

---

### 🛠️ Matériel Requis
* **Microcontrôleur :** Arduino Uno
* **Affichage :** Écran LCD 16x2 avec module I2C (adresse `0x27`)
* **LEDs :** 3 (Vert, Jaune, Rouge)
* **Résistances :** 3x 220 Ohms
* **Câblage :** Breadboard et fils de connexion
